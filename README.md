# Magic Wallet

Open source Ethereum wallet, using Magic.Link for authentication

Tech:
Typescript
NextJS

## Quickstart

Clone this

Instal dependencies
`yarn`

Set up .env file with
`NEXT_PUBLIC_MAGIC_KEY #Magic key`

Run
`yarn dev`

## Deploy to Vercel

Just associate the repo to vercel and add your `NEXT_PUBLIC_MAGIC_KEY`
